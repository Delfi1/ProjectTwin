# Проект "Цифоровой двойник"
## Скрипт для развёртывания и проект на UE 5.
## Требования к железу - видеокарта RTX 3060 или мощнее.

## Как работает? Instant-ngp от Nvidia создаёт грубую модель объекта по видео, требуется ручная доработка.

### Сначала нужно установить все зависимости:
``` pip install -r requirements.txt```

### Чтобы запустить скрипт нам нужна сама нейронка, устанавливаем instant-ngp
``` git clone https://github.com/NVlabs/instant-ngp.git ```

### Скрипт берёт все видео из папки которую нужно указать как аргумент и ищет в ней видео по паттерну t0, t1, t2... 
``` python main.py <НАЗВАНИЕ ПАПКИ>```
